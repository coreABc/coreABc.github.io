Hexo+Github部署

  1. 部署需要的工具(nodejs、git、hexo)
    • nodejs脚本，需要利用npm安装，或者用国内的淘宝镜像用cnpm安装，nodejs安装教程

      原地址和淘宝镜像地址转换方法：

      • npm转换到cnpm
        1、 切换淘宝源
        npm config set registry https://registry.npm.taobao.org --g
        2、 验证配置
        npm config get registry
        3、 切换cnpm
        npm install -g cnpm registry=https://registry.npm.taobao.org
      • cnpm转换到npm
        1、 切换到淘宝cnpm
        npm config set registry https://registry.npm.taobao.org
        2、 验证配置
        npm config get registry
        3、 还原npm
        >npm config set registry https://registry.npmjs.org
    • Git，开源的分布式版本控制系统。安装教程及其环境配置，使用node -v、npm -v/cnpm -v、git --version,git安装完成之后，会在开始菜单中生成Git Bash、Git GUI、Git CMD。
    • hexo下载利用npm install hexo-cli -g或者cnpm install hexo-cli -g，下载完成后用hexo -v查看是否安装成功。
  2. 搭建Github仓库
    进入github的repositry，新建一个仓库，命名方式用户名/用户名.github,io，可勾选Add a README file。
  3. 生成ssh keys
    随便找一个文件夹，右键打开Bash here，输入ssh查看，默认安装了ssh，输入ssh-keygen -t rsa -C "邮箱地址"，会在C://Users/用户名/.ssh/，打开id_rsa.pub，复制内容，然后打开Github的setting,点击SSH，new SSH key,tittle自己取，将刚才复制的内容粘贴到Key中，点击Add。
    检测ssh是否绑定成功:ssh -T git@guthub.com,输入yes，出现hi 你的用户名,you have successfully...即成功，
  4. 生成本地的博客内容
    创建一个文件夹，用于存放博客内容，在开始菜单找到Git Bash以管理员身份运行，进入文件夹，npm install -g hexo安装hexo，然后输入hexo init初始化hexo,出现Starting Blogging with hexo即可。之后hexo s静态生成本地博客页面。
  5. 发布博客到Github上
    将存放博客的文件夹中的_config.yml编辑，在最后一行将deploy修改， 然后在打开Git bash进入博客文件夹，添加部署依赖npm install hexo-deployer-git --save，输入hexo g生成页面，输入hexo d上传，期间会进行Github账号密码的验证，在输入密码的时候，可以在Github创建令牌授权，然后输入即可。最后通过仓库网址访问博客网站你的仓库名.github.io。
    这样hexo的博客网站就初步建立好了，美化插件可以在网上找到。
